# Implementation Plan: Sprint 3 - Secure Docker Sandbox Execution

**Branch**: `003-docker-sandbox` | **Date**: 2026-04-26 | **Spec**: [spec.md](spec.md)
**Input**: Feature specification from `/specs/003-docker-sandbox/spec.md`

## Summary

Extend the LangGraph diagnostic pipeline with an `execute_fix` node that executes
a mock-generated remediation script inside an ephemeral, resource-limited Docker
container (`python:3.10-alpine`), captures stdout/stderr/return-code, and always
cleans up the container. Errors (Docker unavailable, timeout) are captured into
pipeline state rather than raised, so the pipeline always reaches `format_report`.
Sprint 3 delivers mock `fix_script` generation from `analyze_error` and integrates
execution results into the existing markdown report.

**Scale/Scope**: Single-script execution; synchronous LangGraph node; one container per pipeline invocation

## Key Technical Decisions

### 1. Unconditional edge from `execute_fix` to `format_report`

`execute_fix` captures all its errors internally. The graph edge is unconditional — there is no routing logic out of this node. This satisfies FR-007. `_route_after_analyze` returns `"execute_fix"` instead of `"format_report"`.

### 2. `fix_script` returned as top-level key, not nested in `AnalysisResult`

Avoids modifying the established `AnalysisResult` TypedDict schema. `fix_script` is returned alongside `analysis_result` in `analyze_error`'s return dict. Keeping it top-level matches the `parse_error`/`analysis_error` precedent.

### 3. `detach=True` + explicit `container.wait(timeout=5)`

Provides a `Container` handle that can be force-destroyed in `finally` regardless of outcome. `container.wait()` timeout raises `requests.exceptions.ReadTimeout` (not a `docker` SDK exception — uses `requests` under the hood).

### 4. Test-First gate is the first executable phase

No implementation files are created before all tests are written and confirmed failing. This is Constitution Principle III (NON-NEGOTIABLE). Tests must be committed with a message indicating they fail.
